Adding Measures table 12

1. Creating _Measures Table. Go to the "Home" tab and click "Enter Data" and just rename the blank table to "Measures"  and click OK. Create 1 measure at a time.  click on Measures table in the fields section and then table tools ribbon , click 'new measure' and enter 1 measure formula from below and click tick next to it to create that measure. click new measure on the top again to create another measure and do the same process until all below measures are created in '_Measures' table. Some of the measures needs formatting, use the yellow highlighted text to format that measure (it is not part of the DAX)

measures:
1. new vehicle EV QTY YOY = 
VAR CY_NV_EV = sum('new-vehicles-type-area'[battery_electric_number])
VAR PY_NV_EV =
    CALCULATE ( sum('new-vehicles-type-area'[battery_electric_number]), DATEADD('Calendar'[Date],-1,YEAR))
RETURN
    IF( ISBLANK(PY_NV_EV),0, CALCULATE(CY_NV_EV-PY_NV_EV))
[format-decimal with 2 places]

2. new vehicle EV QTY YOY % = 
VAR CY_NV_EV = sum('new-vehicles-type-area'[battery_electric_number])
VAR PY_NV_EV =
    CALCULATE ( sum('new-vehicles-type-area'[battery_electric_number]), DATEADD('Calendar'[Date],-1,YEAR))
RETURN
    IF( ISBLANK(PY_NV_EV),0, DIVIDE((CY_NV_EV-PY_NV_EV), PY_NV_EV))
[format-percentage with 1 place]

3. new vehicle EV share = DIVIDE(sum('new-vehicles-type-area'[battery_electric_number]), sum('new-vehicles-type-area'[total_vehicle_number]),0)
[format-percentage with 4 places]

4. new vehicle EV share YOY = 
VAR CY_NV_EVshare = [new vehicle EV share]
VAR PY_NV_EVshare =
    CALCULATE ( [new vehicle EV share], DATEADD('Calendar'[Date],-1,YEAR))
RETURN
    IF( ISBLANK(PY_NV_EVshare),0, CALCULATE((CY_NV_EVshare-PY_NV_EVshare)))
[format-percentage with 2 places]

5. new vehicle EV share YOY% = 
VAR CY_NV_EVshare = [new vehicle EV share]
VAR PY_NV_EVshare =
    CALCULATE ( [new vehicle EV share], DATEADD('Calendar'[Date],-1,YEAR))
RETURN
    IF( ISBLANK(PY_NV_EVshare),0, DIVIDE((CY_NV_EVshare-PY_NV_EVshare), PY_NV_EVshare))
[format-percentage with 1 places]

6. new vehicles CO2_per_km = 
VAR CY_NV_CO2_per_km = AVERAGE('carbon-new-passenger-vehicles'[co2_per_km])
VAR PY_NV_CO2_per_km =
    CALCULATE ( AVERAGE('carbon-new-passenger-vehicles'[co2_per_km]), DATEADD('Calendar'[Date],-1,YEAR))
RETURN
    FORMAT(IF( ISBLANK(PY_NV_CO2_per_km),0, CALCULATE(CY_NV_CO2_per_km-PY_NV_CO2_per_km)),"0.00g/km")

7. new vehicles CO2_per_km YOY % = 
VAR CY_NV_CO2_per_km = AVERAGE('carbon-new-passenger-vehicles'[co2_per_km])
VAR PY_NV_CO2_per_km =

[format-percentage with 2 places]

8. Renew Consumption share = DIVIDE([Total Renewable Consumption],[Total Energy Consumption],0)
[format-percentage with 1 places]

RETURN
    FORMAT(IF( ISBLANK(PY_renew_consumption),0, CALCULATE(CY_renew_consumption-PY_renew_consumption)),"0.00TWh")
