1. Click on canvas, click card visualizations, drag 'EV QTY YOYIndicator' from'_Measures' to fields.( for 1st indicator)
2. In the format->visual, in callout value, set font size 20, color -> click fx(conditional formatting) set format style= rules, field based on =new vehicle EV QTY YOY %,
value >0 and <=max then #078405
value >=min and <=0 then #AB1409
category label= off
3. In the format->general, set
Height=30
width=42
horizontal=164
vertical=160
title=off
effects-background=off

vertical=321
