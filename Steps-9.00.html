Transforming the first file Data in Power Query Editor 9

  1. Select code column, in the home ribbon click remove columns and select remove columns.
  2. Chage the Entity column name to Country by right clicking on that column and choose Rename.
  3. Select Country column, hold the shift and select year column too , now go to add column ribbon and click merge columns.
  4. Enter ‘_’ as custom separator, ‘Country_Year’ as new column name and click ok.
  5.  Select All 5 vehicle type column numbers by holding shift, in the add column ribbon click statistics and select sum.
  6. A new column is added at the end by summing all 5 columns, rename it as ‘total_vehicle_number’.


7. Sort year ascending by right click and click on year column ascending. (we are getting rid of all years of data before 2013-180 top rows)
8.  In the home ribbon click remove rows, select remove top rows and enter 180 in the box and click ok.
9.  There is only 1 row of data from 2020, to remove that in the home ribbon, click remove rows and select remove bottom rows and enter 1 in the box and click ok.
10.  Now filter out countries that doesn’t have all year’s data. Click arrow button on the country column and deselect ‘Europe’, ‘Iceland’, ‘Turkey’ and click ok.

11.     Now to change ‘year’ to date format, first change year column data type to text. To do that , in the home ribbon, under transform section there is datatype button and click it and select text.

12.     Sort the year column to ascending again (by clicking arrow next to column name and select ascending) , now in the transform ribbon click datatype and select ‘date’.
13.     Now the year is in date format, change that column name to ‘Date’.
14.     Now the data in this table is ready to load, click on file and select close & apply. You will see that table loaded into power bi.
