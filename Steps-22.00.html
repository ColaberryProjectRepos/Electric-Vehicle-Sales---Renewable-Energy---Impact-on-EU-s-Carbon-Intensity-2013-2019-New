CO2 by Country (Scatter Plot - Bottom Left) 22

  1. Click on an empty space on the canvas, and from the visualizations pane select scatter chart.
  2. Fill up fields values= Country; X- axis = Renew Consumption YoY %; Y-Axis = new vehicle EV share YOY%; size=battery_electric_number; Tooltips = new vehicle EV QTY YOY %,new vehicle EV share YOY%, new vehicles CO2_per_km YOY %.
  3. Rename for the visual: new vehicle EV QTY YOY %->EV QTY YoY, new vehicle EV share YOY%=EV share YoY, Renew Consumption YoY %=Renew Consumption YoY,  new vehicles CO2_per_km YOY %=CO2 YoY.
  4. In Format Visual: X-axis: on, Values: Font=Segoe UI (9),color=black, Title : on, Font=Segoe UI (9),color=#A562C9, Y-axis: on, Values: Font=Segoe UI (9),color=black, Title : on, Font=Segoe UI (12),color=#20A49E,  Legend: Position- Top left, Font=Segoe UI (10),color=black, Title: on, title text='CO2  YoY', Gridlines: off, Markers: color: click (fx) and do conditional formatting as below. Category label: on, values: font=segoe UI(9), color= black, background=off.
  5. color conditional formatting: minimum color=#F4EE81, center color=#DDA23C, Maximum color=#D86D10(check screen shot below )
  6. For Format  General: Properties, Size, Height=320, Width=540, Position, Horizontal= 200, Vertical=400. Title=On, Text=CO2 by Country, Font=DIN (15), Text color= #FFFFFF, Background color=#59D8E6 , Horizontal alignment=Center, Effects: Background=On, Background color= #E6E6E6, Transparency=20%.
  7. Drag Year to 'add data field here' in Filters and set filter type = basic filtering, tick select all and deselect 2013.
